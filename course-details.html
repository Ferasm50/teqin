وصف الكورس

هذا الكورس الشامل يغطي جميع جوانب تطوير الويب المتكامل بدءًا من الأساسيات وحتى التقنيات المتقدمة. ستتعلم بناء تطبيقات ويب كاملة باستخدام أحدث التقنيات والأدوات.

الكورس مصمم للمبتدئين والمتوسطين الذين يرغبون في دخول مجال تطوير الويب أو تطوير مهاراتهم الحالية. لا تحتاج إلى خبرة مسبقة في البرمجة، ولكن معرفة أساسية بـ HTML و CSS ستكون مفيدة.

المهارات التي ستكتسبها

تطوير الواجهة الأمامية (Frontend)

React، JavaScript ES6+، HTML5 و CSS3

تطوير الواجهة الخلفية (Backend)

Node.js، Express.js، واجهات برمجة تطبيقات (APIs) RESTful

إدارة قواعد البيانات

MongoDB، Mongoose، نمذجة البيانات

الأمن والاختبار

المصادقة، JWT، اختبار الوحدات

المتطلبات المسبقة

  • فهم أساسي للغة HTML و CSS
  • الإلمام بأساسيات JavaScript (يفضل)
  • جهاز حاسوب متصل بالإنترنت
  • محرر نصوص أو بيئة تطوير متكاملة (IDE) مثبتة
  • رغبة حقيقية في التعلم والتطوير
